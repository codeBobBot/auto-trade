# CLOB API 申请详解 & 替代方案

## CLOB API 申请流程详解

### 为什么需要 CLOB API?

CLOB (Central Limit Order Book) API 是 Polymarket 的官方交易接口，允许：
- ✅ 程序化下单（自动交易）
- ✅ 实时查询账户余额和订单状态
- ✅ 取消/修改订单
- ✅ 获取历史交易记录

### 申请条件

| 要求 | 说明 | 难度 |
|------|------|------|
| 年龄 | 18岁以上 | ⭐ |
| 身份 | 有效身份证件 | ⭐ |
| 地区 | 非受限国家/地区 | ⭐⭐ |
| KYC | 完成身份验证 | ⭐⭐ |
| 资金 | 账户需有资金 | ⭐ |

**受限地区**: 美国部分州、中国大陆、朝鲜、伊朗等

### 详细申请步骤

#### 步骤 1: 注册账号 (5分钟)

1. 访问 https://polymarket.com/
2. 点击 "Sign Up"
3. 输入邮箱地址
4. 设置密码
5. 验证邮箱

#### 步骤 2: KYC 身份验证 (1-24小时)

1. 登录后点击右上角头像 → "Verify Identity"
2. 选择证件类型:
   - 护照 (推荐，通过率高)
   - 身份证
   - 驾照
3. 上传证件照片（正面和背面）
4. 进行人脸识别（自拍视频）
5. 等待审核（通常 1-24 小时）

**提高通过率技巧**:
- 确保光线充足
- 证件四角完整入镜
- 文字清晰可见
- 使用护照比身份证通过率高

#### 步骤 3: 充值账户 (10分钟)

1. 点击 "Deposit"
2. 选择网络:
   - **Polygon** (推荐，手续费低)
   - Ethereum (手续费高)
3. 复制充值地址
4. 从交易所或钱包转账 USDC
5. 等待确认（Polygon 约 5-10 分钟）

**建议充值金额**:
- 测试: $50-100
- 正式运行: $500-2000

#### 步骤 4: 申请 API Key (5分钟)

1. 访问 https://polymarket.com/account/api
2. 点击 "Create API Key"
3. 填写信息:
   - Name: "Trading Bot" (任意名称)
   - Permissions: 勾选 "Read" 和 "Trade"
   - IP Whitelist: 可选，建议填写服务器 IP
4. 点击 "Create"
5. **立即保存**以下信息（只显示一次）:
   - API Key
   - API Secret
   - Passphrase

⚠️ **重要**: API Secret 只显示一次，丢失需重新创建

#### 步骤 5: 配置到系统

编辑 `config/.env`:

```bash
CLOB_API_KEY="your_api_key_here"
CLOB_API_SECRET="your_api_secret_here"
CLOB_API_PASSPHRASE="your_passphrase_here"
```

## 没有 CLOB API 的替代方案

### 方案 1: 手动交易 (推荐过渡方案)

**工作原理**:
```
系统自动分析 → 生成交易信号 → 人工确认 → 手动下单
```

**实施方式**:
1. 系统监控市场并生成交易建议
2. 通过 Telegram 发送通知
3. 您在 Polymarket 网站手动下单
4. 系统记录交易结果

**优点**:
- ✅ 无需 KYC
- ✅ 无需 API
- ✅ 人工把关，降低风险
- ✅ 适合学习和测试

**缺点**:
- ❌ 无法完全自动化
- ❌ 可能错过快速机会
- ❌ 需要人工值守

**适用场景**:
- 初期测试策略
- 资金量较小 ($50-200)
- 学习预测市场

### 方案 3: 信息套利服务化

**商业模式**: 不出金交易，而是出售信号

**实施方式**:
1. 系统持续监控和分析
2. 生成高质量交易信号
3. 通过订阅制出售信号
4. 客户自行在 Polymarket 交易

**盈利方式**:
- 订阅费: $20-50/月
- 信号准确率分成
- 高级分析报告

**优点**:
- ✅ 无需 KYC
- ✅ 无需承担交易风险
- ✅ 可规模化
- ✅ 被动收入

**缺点**:
- ❌ 需要建立客户群
- ❌ 需要维护信誉
- ❌ 收入不稳定
